# Programação Web

## MyJornal:

### Descrição
- Rede social em que os usuários podem escrever artigos e compartilhá-los entre si. A página inicial pede o login, depois que o usuário criar e entrar com sua conta ele pode ler artigos de desconhecidos ou artigos de amigos. Também se quiser publicar uma postagem ele tem a opção de deixá-la pública ou ser visível apenas para amigos. Cada artigo deve ter uma imagem, título e texto.

- O trabalho apresenta  7 paginas, a primeira mostra uma imagem do site e sua logo com botão de Cadastro e Login. Apertando o botão de Cadastro abre um formulário  pedindo um username, e-mail, senha e confirmar senha do usuário. No botão de login aparece só o e-mail e senha.

- Apertando o botão de logar aparece a página principal quando está logado. Nela você pode deslogar, criar uma matéria, alterar o cadastro, adicionar um amigo e uma lista de matérias onde é possível ver seu título e imagem. Na página de criar uma matéria aparece um formulário pedindo, título, texto, imagem e um tipo (se ela é para amigos ou público em geral). No adicionar amigo aparece uma lista de nomes de usuários e você pode clicar para adicioná-lo como amigo. No alterar cadastro você pode alterar qualquer dado de seu cadastro. Quando se abre uma matéria você vê seu título, imagem, texto e quem fez.

-O dados guardados são o usuário com: username, e-mail, senha, lista de amigos. As matérias com: titulo, texto, tipo, imagem, proprietário e por fim relacionamentos com usuário e amigo desse usuario.
