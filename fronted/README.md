# Programação Web

## MyJornal:

### Descrição
- Rede social em que os usuários podem escrever artigos e compartilhá-los entre si. A página inicial pede o login, depois que o usuário criar e entrar com sua conta ele pode ler artigos de desconhecidos ou artigos de amigos. Também se quiser publicar uma postagem ele tem a opção de deixá-la pública ou ser visível apenas para amigos. Cada artigo deve ter uma imagem, título e texto.
